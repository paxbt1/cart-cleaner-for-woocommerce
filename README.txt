== Description ==
Auto Cart Cleaner for WooCommerce is a powerful plugin that enhances your WooCommerce store by providing automatic cart cleaning and real-time stock management. It helps you maintain a clean and efficient cart for your customers while managing your stock effectively.

Key Features:
- **Automatic Cart Cleaning:** Automatic cart cleaning at defined intervals.
- **Real-Time Stock Management:** Efficiently handles stock quantity by adjusting product availability when items are added to or removed from the cart.
- **Enhanced User Experience:** Ensures a seamless shopping experience by preventing cart clutter and stock-related issues.
- **Easy Configuration:** User-friendly settings to define cart cleaning intervals and stock management preferences.
- **WooCommerce Compatibility:** Fully compatible with WooCommerce 4.0 and later versions.


== Installation ==
1. Upload the 'auto-cart-cleaner-for-woocommerce' folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings through the cart-clean-settings panel.

== Frequently Asked Questions ==
= Can I customize the cart cleaning time period? =
Yes, you can define the time period for cart cleaning in the plugin settings.

= How does the stock management work? =
The plugin adjusts stock availability in real-time when items are added to or removed from the cart.

= How to enable stock management ? =
you can simply check the Enable Stock Quantity Sync at the cart-clean-settings panel.
